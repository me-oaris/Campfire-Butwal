## 🚀 About the Game

**THE LAST OPTION** is a fast-paced, decision-driven game built from scratch during a **24-hour Game Jam at CAMPFIRE-BUTWAL**.

In just one intense day, our team brainstormed, designed, and developed a complete playable game — turning an idea into reality under extreme time pressure.

---

## 🕹️ Play the Game

👉 **https://oaris.itch.io/thelastoption**

---

## 🧠 Game Concept

In *The Last Option*, players are placed in high-stakes situations where every second matters.

⚔️ Make quick decisions  
🧩 Face challenging scenarios  
🎯 Choose wisely — because there’s always **one last option**

A wrong move could cost everything.

---

## ⏱️ Built in 24 Hours

🕒 This game was created within a strict **24-hour time limit**, which means:

- No pre-built systems  
- No long planning phase  
- Pure execution and creativity  

💡 From idea → prototype → final game  
**All completed in ONE day.**

---
